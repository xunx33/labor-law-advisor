# 劳动法律文书模板

> 模板中的 `[方括号内容]` 需根据用户实际情况替换。如用户未提供完整信息，先补充信息收集再填充，也可留空或占位符显示。

## 1. 被迫解除劳动合同通知书

```
被迫解除劳动合同通知书

致：[用人单位全称]

本人 [姓名]，身份证号 [号码]，于 [入职日期] 入职贵单位，担任 [岗位]。

因贵单位存在以下违法行为：
□ 未及时足额支付劳动报酬（具体：[拖欠金额及月份]）
□ 未依法为本人缴纳社会保险费
□ 未按照劳动合同约定提供劳动保护或劳动条件
□ 未及时足额支付加班费
□ 规章制度违反法律、法规规定，损害劳动者权益
□ 其他：[具体说明]

依据《中华人民共和国劳动合同法》第三十八条之规定，本人现通知贵单位：
自本通知书送达之日起，解除双方劳动合同关系。

请贵单位于接到本通知后 [X] 日内：
1. 为本人办理离职手续，出具解除劳动合同证明；
2. 支付拖欠的劳动报酬共计 [金额] 元；
3. 支付经济补偿金共计 [金额] 元（[工作年限] 年 × [月平均工资] 元）；
4. 补缴社会保险费。

特此通知。

通知人：[签名]
日期：[日期]

送达方式：□ EMS邮寄（单号：） □ 当面送达（见证人：） □ 电子邮件
```

## 3. 确认劳动关系仲裁申请书

```
确认劳动关系仲裁申请书

申请人：[姓名]，身份证号 [号码]，住址 [地址]，电话 [电话]
被申请人：[单位全称]，地址 [单位地址]，法定代表人 [姓名]

仲裁请求：
请求确认申请人与被申请人之间存在劳动关系。

事实与理由：
申请人于 [入职日期] 起在被申请人处工作，担任 [岗位]，月工资 [金额] 元。工作期间，申请人接受被申请人的管理，从事被申请人安排的有报酬的劳动。

但被申请人未与申请人签订书面劳动合同。

申请人与被申请人之间符合原劳动和社会保障部《关于确立劳动关系有关事项的通知》（劳社部发〔2005〕12号）第一条规定的劳动关系成立要件：
（一）用人单位和劳动者符合法律、法规规定的主体资格；
（二）用人单位依法制定的各项劳动规章制度适用于劳动者，劳动者受用人单位的劳动管理，从事用人单位安排的有报酬的劳动；
（三）劳动者提供的劳动是用人单位业务的组成部分。

为证明上述事实，申请人提交以下证据：
[列举证据]

综上，请求贵委依法确认申请人与被申请人之间存在劳动关系。

此致
[XX 劳动人事争议仲裁委员会]

申请人：[签名]
日期：[日期]
```

## 10. 竞业限制补偿金确认/催告函

```
竞业限制经济补偿金催告函

致：[公司名称]

本人 [姓名]，身份证号 [号码]，原系贵公司 [部门] [岗位] 员工。

本人与贵公司于 [日期] 签订了竞业限制协议，约定竞业限制期限为离职后 [X] 个月，贵公司按月支付竞业限制经济补偿金 [金额] 元。

本人已于 [离职日期] 从贵公司离职，但截至本函发出之日：
□ 贵公司尚未支付竞业限制经济补偿金
□ 贵公司已连续 [X] 个月未支付竞业限制经济补偿金

依据《最高人民法院关于审理劳动争议案件适用法律问题的解释（一）》第三十七条、第三十八条之规定：
- 当事人约定了竞业限制，用人单位应支付经济补偿
- 用人单位3个月未支付经济补偿的，劳动者可请求解除竞业限制约定

现本人特此催告贵公司：
1. 于收到本函后 [X] 日内支付拖欠的竞业限制经济补偿金共计 [金额] 元；
2. 如逾期仍未支付，本人将依法解除竞业限制约定，不再受竞业限制约束。

特此函告。

催告人：[签名]
日期：[日期]
```
